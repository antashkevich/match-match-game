  • high (8 x 3)

how to play:

memory is a counter game where the object is to find pairs.

when the game begins, all pictures are hidden.

to play:

  • Record the information about yourself and start playing
  • select two cards
  • if the cards are the same, they will disappear, and you will be able to choose two other cards
  • if the cards are different, they will return to their original position and you will be able to select any two cards again
  • The player that finds all pairs wins!
  • Have Fun!
